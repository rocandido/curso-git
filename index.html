GIT

Trabalhando em Equipe com Controle e Segurança

Um curso que explora os benefícios de utilizar Git como ferramenta de controle de versão par projetos em qualquer linguagem, em qualquer plataforma.

Principais benefícios

  • Funcniona de maneira distribuída
  • Permite a edição concorrente de arquivos do projeto
  • Não depeende de uma conexão com o servidor
